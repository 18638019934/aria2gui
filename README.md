aria2gui
===========

![UI](http://i.imgur.com/Jq8Ttyc.png)

## Features:

- 支持百度，115，360等网盘的导出（需要浏览器插件支持）
- 集成了aria2，运行后即完成配置工作

## Usage:

- 解压后拖到应用里面运行
- 浏览器导出aria2
